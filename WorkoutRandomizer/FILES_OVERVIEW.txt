│  STEP 2: Add Privacy Descriptions                                        │
│  ─────────────────────────────────────────                               │
│  Add to iOS Info.plist:                                                  │
│    <key>NSHealthShareUsageDescription</key>                              │
│    <string>Display workout metrics during routines</string>              │
│    <key>NSHealthUpdateUsageDescription</key>                             │
│    <string>Save workout data to HealthKit</string>                       │

│  Add to watchOS Info.plist:                                              │
│    <key>NSHealthShareUsageDescription</key>                              │
│    <string>Track heart rate and calories</string>                        │
│    <key>NSHealthUpdateUsageDescription</key>                             │
│    <string>Save workout sessions</string>                                │

│  Q: What if metrics don't show up?                                       │
│  A: Check:                                                                │
